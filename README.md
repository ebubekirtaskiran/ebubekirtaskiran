# Ebubekir Taşkıran

Yazılım Mühendisliği öğrencisiyim. **Bakımı kolay, güvenli ve ölçeklenebilir yazılım sistemleri**
geliştirmeye odaklanıyorum. İlgi alanlarım özellikle **yazılım mimarisi**, **güvenli API entegrasyonu**
ve **bilgisayarlı görü / OCR tabanlı uygulamalar** üzerinedir.

## Odaklandığım Alanlar
- **Yazılım Mimarisi:** MVC, modüler yapı, sorumlulukların ayrılması
- **Güvenli Yapılandırma:** `.env`, environment variable kullanımı, gizli bilgilerin Git’e alınmaması
- **Bilgisayarlı Görü / OCR:** görüntüden metin ve matematiksel ifade tanıma
- **Algoritmalar & Problem Çözme:** veri yapıları, performans odaklı yaklaşım
- **Sistem Düşüncesi:** sayısal mantık, donanıma yakın yazılım bakış açısı
